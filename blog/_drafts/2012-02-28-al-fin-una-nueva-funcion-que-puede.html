Hoy me he despertado con mi navegador pidendome que le de permiso para usar Gmail como sistema para abrir las direcciones mail y componer para enviar y con esa noticia:

Open email links directly in Gmail using Chrome

Como he dicho, para tener esta funcionalidad, hasta ahora, tenías que instalarte un pequeño programa de dudosa seguridad que hacía lo mismo que ahora hace el HTML5 que se está incorporando poco a poco en todos los sitios web y al que los exploradores de internet se están adaptando para sacarle el mayor beneficio posible.

Como dice al final del post, esta funcionalidad también está para aquellos que usen Firefox o explorer. Por supuesto, es de esperar (estoy a la espera de confirmarlo) que funcionará también en linux, sin necesidad también de instalar, por ejemplo Gnome Gmail, por poner un ejemplo.
