El código QR es una tecnología generalmente para smartphones que consiste en un código que captas con tu cámara y un software descifra el contenido: una página web, una canción, un texto, un email, etc.

Pero no todos los smartphones tienen este software y aunque esto de los códigos está muy extendido apenas hay gente que tiene acceso. Pero ahora ha salido una aplicación para Google Chrome que te permite leer estos códigos.



QRreader beta es una extensión para chrome que permite pinchar en la imagen y elegir si queremos que descifre el código QR.
